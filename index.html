Instructions:

1. Select a Grid Height and Width size, then click submit to create the grid.

2. Next, in "Pick A Color" click the color block to select a color.

3. Click on a cell on the canvas to add color; to change colors, select color block again.

4. To erase the color in a cell or cells, just click those cell again.
